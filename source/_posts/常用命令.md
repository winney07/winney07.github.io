#### 重装系统后

##### 1. git命令窗口要重新输入用户名

```
git config --global user.name "Github用户名"       //自己Github的账号名
git config --global user.email "Github邮箱"        //自己注册Github的邮箱地址
```

##### 2. 重新设置SSH

1. 在docs命令窗口，输入以下命令：

   ```
   ssh-keygen -t rsa -C  "Github邮箱"        //自己注册Github的邮箱地址
   ```

2. 在C:\Users\admin\\.ssh目录下（根据你自己电脑用户名，打开对应的目录）

3. 找到id_rsa.pub文件

4. 在GitHub个人设置中创建一个SSH Keys ，将id_rsa.pub里面的内容复制到这里面。

5. 重新输入命令，即可上传文件

[生成SSH KEY](https://winney07.github.io/2018/08/01/Hexo-GitHub-Pages-%E6%90%AD%E5%BB%BA%E4%B8%AA%E4%BA%BA%E5%8D%9A%E5%AE%A2/)

百度云盘

- 在线观看

  1. 电脑端

     火狐浏览器打开云盘网页版——打开视频后——右键——选择播放速度

     下载ES文件浏览器+MX播放器（2个软件配合使用)

     > ES文件浏览器是一款功能强大免费的本地和网络文件管理器。
     > 号称安卓平台上的第三方文件管理工具。
     > 主要的三大功能是
     > 文件管理、多媒体浏览和云储存，支持海量的网盘空间存放数据。
     > 下图为倍速播放操作流程
     > 首先打开ES文件浏览器——网盘——百度云盘(登录自己的账号）——选中一个视频一选择MX播放器——全屏状态下左上角有倍速选择，点击后在右下角可以输入变速数值。
