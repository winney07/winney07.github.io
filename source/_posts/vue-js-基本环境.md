#### 1、安装有npm
{% link node.js官网  https://nodejs.org/en/ %}
#### 2、安装vue-cli
```bash
npm i vue-cli -g

输入vue命令查看是否安装成功
输入vue list 查看
```
<!--more-->
#### 3、建立工程
```bash
vue init webpack 项目名
```
#### 4、进入 项目名 目录，安装脚手架项目的基本支持包
```bash
npm i
```
#### 5、运行基本Vue.js程序
```bash
npm run dev
```
