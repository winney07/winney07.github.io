React Native的优点

  • 开发体验好
    • 用统一的代码规范开发移动端程序，不用关注移动端的差异
  • 开发成本低
    • 开发一次，可以生成Android和iOs两个系统上的App- Learn once, write anywhere
  • 学习成本低
    • 只要掌握JavaScript和React，就可以进行移动端开发了

React Native的不足

  • 不成熟;
    • 项目版本更新维护较频繁，学习成本高;
    • 试错成本高，有些问题较少解决方案，易耽误开发进度。
  • 性能差
    • 整体性能仍不如原生
  • 兼容性差
    • 涉及底层的功能，需要针对Android和iOs双端单独开发;
