## 常用的特殊符号

| 特殊符号 | HTML写法  | 描述       |
| -------- | --------- | ---------- |
|          | `&nbsp;`  | 空格       |
| &        | `&amp;`   | 和、与     |
| "        | `&quot;`  | 英文双引号 |
| '        | `&apos;`  | 英文单引号 |
| >        | `&gt;`    | 大于       |
| ≥        | `&ge;`    | 大于等于   |
| ＜       | `&lt;`    | 小于       |
| ≤        | `&le;`    | 小于等于   |
| ≠        | `&ne;`    | 不等于     |
| ¥        | `&yen;`   | 人民币符号 |
| &infin;  | `&infin;` | 无穷       |
